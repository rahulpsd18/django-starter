Welcome to Django-Starter.

Replace django-starter with the name of your-app using "Find in Project" or manually from the following files:

  • settings.py - LINE 43: ROOT_URLCONF = 'django-starter.urls'
  • settings.py - LINE 62: WSGI_APPLICATION = 'django-starter.wsgi.application'
  • wsgi.py - LINE 11: os.environ.setdefault("DJANGO_SETTINGS_MODULE", "django-starter.settings")
  • manage.py - LINE 6: os.environ.setdefault("DJANGO_SETTINGS_MODULE", "django-starter.settings")
  • Procfile - LINE 1: web: gunicorn django-starter.wsgi --log-file -
  1. Create new app using $ python manage.py startapp <app-name>
  2. cd <app-name> && mkdir static && mkdir templates
  3. And on you go. (Don't forget to add it to your settings.py - INSTALLED_APPS)
Production Mode
TIP: Run python manage.py collectstatic manually while working in Production mode on localhost.
  1. Set DEBUG = False in settings.py
  2. Set your SECRET_KEY in settings.py
  3. Add hosts accordingly in ALLOWED_HOSTS= [] in settings.py
